About DR-CRUD

DR-CRUD or Data Repository/Create Read Update Delete is a showcase/playground for my web development skills with the stack: HTML, CSS through tailwind, and Vuejs as frontend, and PHP through Laravel Framework and SQL through MySQL as backend, with Inertia.js as a bridge between front and backend. This site is the frontend, for the read component of CRUD, Another site will be the backend, for the create, update, and delete components of CRUD, with the backend connecting the two.
